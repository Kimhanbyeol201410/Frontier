> **범위:** 채팅에 메시지 시각이 없어, “오늘 12시 이후”를 시스템에서 자동으로 잘라내지는 못했습니다. 아래는 **이번 작업 세션에 나타난 요청**과, 시스템에 전달된 **이전 대화 요약**에 적힌 내용을 구분해 두었습니다. 12시 이전·이후를 나누려면 직접 가리켜 주시면 그에 맞춰 문서를 수정하면 됩니다.

## 1. 이번 스레드에서 직접 요청된 항목

### 1-1. 「목숨을 걸어」카드·파워 동작 변경

**요청 요지**

- 턴 시작 시 “보유 힘·민첩·열기만큼 다시 얻는” **2배(턴 시작) 효과 제거**
- **첫 사용 보너스가 아니라**, 카드를 **사용할 때마다** 힘·민첩·열기 부여  
  - 기본: 힘 1, 민첩 1, 열기 10  
  - 강화: 힘 2, 민첩 2, 열기 20

**처리 방향 (요약에 따른 구현)**

- `FrontierSession`: 플레이당 보너스 저장·조회
- `BetYourLifeCard`: 플레이 시 보너스 기록, 강화 시 수치 증가
- `ShumitBetYourLifePower`: 턴 시작 훅 제거, 카드 플레이 시마다 적용
- `kor`/`eng` 카드·파워 로컬라이즈 문구 정리
